# Frontend

Frontend scaffold for the AI-first HCP CRM.

## Planned Responsibilities
- Render the split-screen CRM layout
- Show a read-only interaction form on the left
- Show the AI assistant chat on the right
- Surface tool calls and state updates from the backend
- Manage UI state with Redux Toolkit

## UI Structure
- `src/app/`: app shell
- `src/components/`: shared UI pieces
- `src/features/chat/`: assistant chat UI
- `src/features/interaction-form/`: read-only form UI
- `src/store/`: Redux store
- `src/styles/`: global and layout styles
